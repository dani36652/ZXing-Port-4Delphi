This repository makes use of the "zxing-android-embedded" project by
"journeyapps", and proper credit is given to its author. 
The project can be accessed at the following link: 
[https://github.com/journeyapps/zxing-android-embedded](https://github.com/journeyapps/zxing-android-embedded). 
Likewise, this example includes the port of the ZXingCore library available at:
[https://mvnrepository.com/artifact/com.google.zxing/core](https://mvnrepository.com/artifact/com.google.zxing/core),
which allows generating barcodes or QR codes from Delphi.
